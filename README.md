# IPL Data Analytics Project

## Overview

This project was developed as part of the BuildX Analytics Bootcamp. It demonstrates the complete data analytics workflow by cleaning, analyzing, and visualizing the IPL Ball-by-Ball Dataset (2008–2025). The project combines Python for data preprocessing, SQL for analytical queries, and Power BI for creating an interactive dashboard that presents meaningful insights from the dataset.

## Dataset

IPL Ball-by-Ball Dataset (2008–2025)
The dataset contains detailed information for every ball bowled in IPL matches, including batting and bowling statistics, match details, venues, seasons, players, and match results.

## Why I Chose This Dataset
I selected this dataset because it offers a large volume of real-world sports data with diverse attributes. It provided an opportunity to practice data cleaning, SQL querying, and dashboard development while exploring player performances, team statistics, and match trends.

## Business Questions Addressed
Which teams have scored the highest number of runs across the IPL seasons?

Who are the leading run scorers and wicket takers in the dataset?

How do team performances vary across different seasons and venues?

## Dashboard Highlights

Top run scorers

Top wicket takers

Interactive slicers and page-level filters

DAX calculated column and measure

Key insights based on the analysis

## Key Insight
The analysis shows that most scoring events in the IPL come from singles and doubles rather than boundaries, emphasizing the importance of consistent strike rotation. It also highlights noticeable differences in scoring across venues and teams.

## Conclusion
This project demonstrates an end-to-end analytics workflow, starting from raw data cleaning and preparation to SQL-based analysis and interactive dashboard creation. It helped strengthen practical skills in Python, SQL, Power BI, and GitHub while working with a real-world sports dataset.
